Solutions des travaux pratiques

Modèles QGIS

Ces fichiers au format .model3 contiennent des modèles de traitement QGIS qui automatisent des séquences d'opérations géospatiales. Ils peuvent être importés dans QGIS pour reproduire facilement les analyses présentées pendant l'atelier.

Comment utiliser un modèle

  1. Ouvrez QGIS
  2. Allez dans le menu Traitement > Boîte à outils de traitements
  3. Cliquez sur l'icône "Ouvrir un modèle existant"
  4. Sélectionnez le fichier .model3 téléchargé

Modèles disponibles

  • 2.1 Analyse du tracé de la fibre optique - trace_de_fibre.model3
  • 2.2 Analyse de la couverture mobile - couverture_mobile.model3
  • 2.3 Analyse de la population et de la demande - demande.model3
  • 2.4 Analyse de la couverture fixe - couverture_fixe.model3

Cahiers Python (Google Colab)

  • 2.4 Analyse de la couverture fixe - couverture_fixe.ipynb - Afin d'ouvrir ce cahier Python avec Google Colaboratory, cliquez sur les trois points à droite et cliquez sur Ouvrir avec Google Colaboratory. Il vous faudra un compte Google (par éxemple, Gmail) afin d'ouvrir ce fichier.
